MOSAIC

Mosaic is an open source grid framework inspired by and similar to Masonry, but with the added ability of relatively positioning tiles (or elements) within your grid. It includes both a CSS and Javascript library, but each can be used on their own (although they pair very well).
SKILLS: JAVASCRIPT, HTML, CSS, SCSS

ELEMENTS

I worked with three other teammates on a virtual reality "element-bending" simulator. With an Oculus Rift and a Myo armband on each arm, the user can make gestures to throw rocks, move water, catapult fireballs, and blast air in a virtual environment.
HackPrinceton 2014 Awards: Best Use of Unity, Best Myo Hack.
SKILLS: UNITY, C#

LITESUGGEST

A lightweight jQuery plugin for suggesting answers in form inputs. Not to be used for large, autocorrect-style suggestions, but rather for suggesting answers in a targeted form. Allows for fast client-side suggestions on websites without a server. Includes customizable CSS styling.
SKILLS: JAVASCRIPT, HTML, CSS
